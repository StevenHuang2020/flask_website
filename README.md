# flask website
A exapmle of blog website developed by [Flask](https://flask.palletsprojects.com/en/2.0.x/).


### Usage
1. Install requirement libraries </br>
```$ pip install -r requirements.txt```
2. Initial database  </br>
```$ python install.py```
3. Start server  </br>
```$ python run.py```
