# flask website
A exapmle of blog website developed by [Flask](https://flask.palletsprojects.com/en/2.0.x/).


### Usage
1. Install requirement libraries </br>
```$ pip install -r requirements.txt```
2. Initialize the database  </br>
```$ python steup_db.py```
3. Start server  </br>
```$ python run.py```
